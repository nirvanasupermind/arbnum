# arbnums
Arbitrary-precision (bignum) ints and floats in Javascript. Currently WIP.
# API
## ArbInt
### constructor()
The ArbInt constructor. Passes in one argument, `str`,the textual representation of the number.

## add()
Adds two ArbInts.

## plus()
Alias for `add()`.

## sub()
Subtracts two ArbInts.
